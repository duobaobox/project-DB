## 📝 基础 Markdown 功能

在学习高级功能前，先快速回顾一下基础格式。

### 文本格式化

```markdown
**粗体文字**
_斜体文字_
**_粗斜体文字_**
~~删除线文字~~
```

效果：
**粗体文字**, _斜体文字_, **_粗斜体文字_**, ~~删除线文字~~

### 列表

无序列表（使用 `-`、`*` 或 `+` 符号）:

```markdown
- 项目 1
- 项目 2
  - 子项目 2.1 (只需在前面多加两个空格)
  - 子项目 2.2
```

效果：

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2

有序列表（使用数字加点）:

```markdown
1. 第一步
2. 第二步
3. 第三步
```

效果：

1. 第一步
2. 第二步
3. 第三步

### 链接和图片

```markdown
[链接文本](https://example.com)
![图片描述](https://via.placeholder.com/600x300)
```

效果：
[链接文本](https://example.com)

![图片描述](https://via.placeholder.com/600x300)

### 代码块

行内代码（使用反引号）:

```markdown
`const example = "hello world";`
```

效果：`const example = "hello world";`

代码块（使用三个反引号包围）:

````markdown
```javascript
function hello() {
  console.log("Hello, world!");
  return true;
}
```
````

效果：

```javascript
function hello() {
  console.log("Hello, world!");
  return true;
}
```

> 💡 **小提示**：在三个反引号后面写上语言名称（如 javascript），可以实现代码高亮哦！

## ✨ 高级功能

下面我们一起了解新增的高级功能，这些功能可以让你的文档更加生动、美观！

### ✅ 任务列表

你可以创建带有复选框的任务列表，非常适合表示待办事项。

```markdown
- [ ] 未完成任务
- [x] 已完成任务
- [ ] 另一个任务
```

效果：

- [ ] 未完成任务
- [x] 已完成任务
- [ ] 另一个任务

> 💡 **使用提示**：`- [ ]` 表示未勾选的任务，`- [x]` 表示已勾选的任务。方括号内必须有一个空格或字母 x。

### 📊 表格

你可以使用简单的语法创建表格，用 `|` 分隔列，用 `-` 分隔表头。

```markdown
| 名称   | 说明       | 数量 |
| ------ | ---------- | ---- |
| 项目 A | 这是项目 A | 10   |
| 项目 B | 这是项目 B | 20   |
| 项目 C | 这是项目 C | 30   |
```

效果：
| 名称 | 说明 | 数量 |
|------|------|------|
| 项目 A | 这是项目 A | 10 |
| 项目 B | 这是项目 B | 20 |
| 项目 C | 这是项目 C | 30 |

> 💡 **排版技巧**：表格的竖线不需要对齐也能正常显示，但对齐会让你的源码更整洁。表头下面的分隔线至少需要三个短横线 `---`。

### 📝 脚注

脚注可以在不打断正文的情况下添加额外信息或参考资料。

```markdown
这是一个带有脚注的句子[^1]。

[^1]: 这是脚注的内容，可以包含多行文本。
```

效果：
这是一个带有脚注的句子[^1]。

[^1]: 这是脚注的内容，可以包含多行文本。

> 💡 **使用说明**：脚注标记可以使用数字或文字（如 `[^note1]`），但必须以 `^` 开头。脚注内容可以放在文档的任何位置，建议统一放在文档底部。

### 🔠 上标和下标

你可以使用上标和下标来表示数学公式或化学式。

```markdown
上标例子：X^2^
下标例子：H~2~O
```

效果：
上标例子：X^2^
下标例子：H~2~O

> 💡 **提示**：使用 `^` 包围的内容会变成上标，使用 `~` 包围的内容会变成下标。

### 📌 自定义容器

自定义容器可以让你的重要信息更加醒目。目前支持三种类型：提示、警告和危险。

```markdown
::: tip
这是一个提示信息。
:::

::: warning
这是一个警告信息。
:::

::: danger
这是一个危险警告。
:::
```

效果：
::: tip
这是一个提示信息。
:::

::: warning
这是一个警告信息。
:::

::: danger
这是一个危险警告。
:::

> 💡 **使用格式**：使用 `::: 类型名称` 开始，内容写在中间，用 `:::` 结束。类型名称目前支持 tip、warning 和 danger。

### 💬 引用块

引用块可以突出显示引用的内容或重要说明。

```markdown
> 这是一个引用块
>
> 它可以包含多个段落
```

效果：

> 这是一个引用块
>
> 它可以包含多个段落

> 💡 **创建方法**：在每行前面加上 `>` 符号和一个空格。如果引用内容有多个段落，段落之间的空行也需要有 `>` 符号。

### 🌐 HTML 内联支持

你可以在 Markdown 中直接使用 HTML 代码，实现更复杂的排版效果。

```markdown
<div style="padding: 15px; background-color: #f0f0f0; border-radius: 5px;">
  <h3>HTML 内容</h3>
  <p>这是通过 HTML 直接编写的内容</p>
</div>
```

效果：

<div style="padding: 15px; background-color: #f0f0f0; border-radius: 5px;">
  <h3>HTML 内容</h3>
  <p>这是通过 HTML 直接编写的内容</p>
</div>

> 💡 **注意事项**：使用 HTML 可以实现更复杂的排版，但可能需要一些 HTML 和 CSS 的基础知识。如果你是完全的小白，建议先掌握其他 Markdown 功能。

### 😊 表情符号支持

你可以在文档中添加表情符号，让内容更生动有趣。

```markdown
常用表情: :smile: :heart: :thumbsup: :rocket: :computer:
```

效果：
常用表情: :smile: :heart: :thumbsup: :rocket: :computer:

> 💡 **使用方法**：用冒号包围表情符号的名称，例如 `:smile:`。目前支持大量常用表情符号。

### 🎨 属性扩展

你可以为内容添加自定义类名和样式，实现更个性化的显示效果。

```markdown
这段文字会变成红色 {.red-text style="color: red;"}
```

效果：
这段文字会变成红色 {.red-text style="color: red;"}

> 💡 **使用格式**：在文本后面用花括号 `{}` 包围属性。`.red-text` 是添加类名，`style="color: red;"` 是直接添加 CSS 样式。

### 🔗 自动链接

系统会自动识别网址并转换为可点击的链接。

```markdown
自动识别网址: https://example.com
```

效果：
自动识别网址: https://example.com

> 💡 **使用提示**：你不需要使用 `[](https://...)` 格式，直接输入网址就会被自动转换为链接。
