# 超简单文档排版指南 - 小白也能学会！

不需要任何编程知识，只需要几个简单的符号，就能让你的项目描述变得美观大方！这份指南告诉你如何美化你的项目文档。

## 📝 必填的开头信息

每个项目文件最开始**必须**有以下内容（不要漏掉开头和结尾的三个横线）：

```
---
id: 你的项目ID（英文字母和横杠，例如my-project）
title: 你的项目标题
description: 简短的项目描述（显示在卡片上）
image: img/你的图片.jpg（项目封面图片路径）
---
```

👆 这部分信息会自动生成项目卡片，一定要填写完整哦！

## 🧭 添加自动生成的目录

想要为长文档添加导航目录？只需在合适的位置添加以下代码：

```
[[toc]]
```

效果如下：

[[toc]]

这将自动生成一个基于文档标题的目录，方便读者快速跳转到感兴趣的部分。

## 🔤 文字样式

想让文字看起来更有重点？试试这些简单的格式：

**粗体文字**：在文字两边加上 `**` 符号  
例如：`**这是粗体**` → **这是粗体**

_斜体文字_：在文字两边加上 `*` 符号  
例如：`*这是斜体*` → _这是斜体_

**_粗斜体_**：在文字两边加上 `***` 符号  
例如：`***这是粗斜体***` → **_这是粗斜体_**

~~删除线~~：在文字两边加上 `~~` 符号  
例如：`~~这是删除线~~` → ~~这是删除线~~

`代码样式`：在文字两边加上 `` ` `` 符号（键盘上数字 1 左边的按键）  
例如：`` `这是代码样式` `` → `这是代码样式`

## 📊 添加标题

你可以使用 `#` 符号来创建不同大小的标题：

```
# 一级标题（最大）
## 二级标题（常用）
### 三级标题
#### 四级标题（较小）
```

👉 记住：标题前的 `#` 和文字之间要有一个空格！

## 📋 制作列表

### 普通列表（不带数字）：

只需在每行前面加上 `-` 或 `*` 再加一个空格：

```
- 第一点
- 第二点
- 第三点
```

显示效果：

- 第一点
- 第二点
- 第三点

### 数字列表：

在每行前面加上数字、小数点和空格：

```
1. 第一步
2. 第二步
3. 第三步
```

显示效果：

1. 第一步
2. 第二步
3. 第三步

### 嵌套列表（列表中的子项）：

只需在子项前多加两个空格：

```
- 主要任务
  - 子任务1
  - 子任务2
- 另一个主要任务
```

显示效果：

- 主要任务
  - 子任务 1
  - 子任务 2
- 另一个主要任务

### 任务列表（带复选框的列表）：

使用方括号来创建带有复选框的列表：

```
- [ ] 未完成的任务
- [x] 已完成的任务
- [ ] 另一个待办事项
```

显示效果：

- [ ] 未完成的任务
- [x] 已完成的任务
- [ ] 另一个待办事项

👉 **提示**：方括号内的空格表示未完成，字母 `x` 表示已完成。

## 💬 添加引用

想要突出一段话？在段落前面加上 `>` 符号和空格：

```
> 这是一段引用的内容。
> 可以占据多行。
```

显示效果：

> 这是一段引用的内容。
> 可以占据多行。

## 📊 制作表格

表格看起来复杂，但创建非常简单：

```
| 姓名 | 年龄 | 职业 |
|------|------|------|
| 张三 | 25 | 设计师 |
| 李四 | 30 | 开发者 |
| 王五 | 28 | 产品经理 |
```

显示效果：

| 姓名 | 年龄 | 职业     |
| ---- | ---- | -------- |
| 张三 | 25   | 设计师   |
| 李四 | 30   | 开发者   |
| 王五 | 28   | 产品经理 |

👉 **提示**：

1. 使用 `|` 符号分隔列
2. 表头下方的 `---` 行是必须的
3. 表格各列的对齐不必完美，只要基本格式正确即可显示

## 📷 插入图片

图片格式很简单：

```
![图片说明文字](img/图片名称.jpg)
```

例如：`![我的项目截图](img/演示图片.jpg)`

效果：

![我的项目截图](img/演示图片.jpg)

👉 **重要提示**：

1. 确保你的图片已放入网站的 `img` 文件夹
2. 图片名称要完全一致（包括大小写）
3. 别忘了图片格式后缀（.jpg、.png 等）

## 🔗 添加链接

链接的格式也很简单：

```
[链接显示的文字](https://网址)
```

例如：`[访问我的博客](https://example.com)` → [访问我的博客](https://example.com)

### 自动链接：

如果你直接输入网址，系统会自动将其转换为可点击的链接：

例如：`https://example.com` → https://example.com

## 📏 添加分隔线

想要在段落之间加一条横线？只需要输入三个连续的横杠：

```
---
```

显示效果：

---

## 💻 代码块

展示多行代码时，使用三个反引号（` ``` `）包围代码：

````
```
// 这是一段代码
function hello() {
  console.log("Hello!");
}
```
````

显示效果：

```
// 这是一段代码
function hello() {
  console.log("Hello!");
}
```

### 代码高亮：

在第一行反引号后添加语言名称，可以启用代码高亮：

````
```javascript
// 这是带高亮的代码
function hello() {
  console.log("Hello!");
  return true;
}
```
````

显示效果：

```javascript
// 这是带高亮的代码
function hello() {
  console.log("Hello!");
  return true;
}
```

## 📝 特殊格式

### 脚注：

你可以使用脚注来添加参考信息：

```
这是带有脚注的句子[^1]。

[^1]: 这是脚注内容，会显示在文档底部。
```

效果：
这是带有脚注的句子[^1]。

[^1]: 这是脚注内容，会显示在文档底部。

### 上标和下标：

```
水的化学式是 H~2~O，爱因斯坦公式是 E=mc^2^
```

效果：
水的化学式是 H~2~O，爱因斯坦公式是 E=mc^2^

## 📢 信息提示框

使用特殊语法可以创建醒目的提示框：

```
::: tip
这是一个提示信息框。
:::

::: warning
这是一个警告信息框。
:::

::: danger
这是一个危险警告框。
:::
```

效果：

::: tip
这是一个提示信息框。
:::

::: warning
这是一个警告信息框。
:::

::: danger
这是一个危险警告框。
:::

## 😊 表情符号

你可以在文档中添加表情符号，使内容更加生动：

```
我很喜欢这个项目 :heart: :smile: :thumbsup:
```

效果：
我很喜欢这个项目 :heart: :smile: :thumbsup:

## 🎯 小技巧

1. **使用表情符号**增加趣味性：在标题前加上简单的 emoji，比如 📊 📝 🔍

2. **段落间空行**：想要新起一个段落？只需要空一行即可

3. **排版整洁**：使用统一的格式，让内容层次分明

4. **预览效果**：刷新网站即可看到你的更改效果

5. **样式属性**：可以为文本添加自定义样式
   ```
   这段文字会变成红色 {.special style="color: red;"}
   ```
   效果：这段文字会变成红色 {.special style="color: red;"}

## 🔍 常见问题解答

**问：为什么我的格式没有生效？**  
答：检查你的符号是否为英文输入法下的符号，中文符号会导致格式失效

**问：图片显示不出来怎么办？**  
答：确保图片已放入 img 文件夹，且图片路径和名称完全正确

**问：如何让我的项目描述看起来更专业？**  
答：使用清晰的标题结构，添加图片展示，使用列表条理化呈现信息

**问：特殊格式（如表情符号、特殊容器）不显示怎么办？**  
答：确保你的文件已保存，并且网站已刷新。如果仍有问题，可能是当前不支持这些功能
